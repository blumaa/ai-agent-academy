# React Todo App with Theme Toggle

Build a React todo application with the following features:

- Add, complete, and delete todo items
- Light/dark mode toggle that persists across page reloads
- Clean component architecture (separate components for list, item, input, theme toggle)
- State management without external libraries (React context or prop drilling — your call)
- Styled with Tailwind CSS

Constraints: Single `src/` directory, no backend, no external state libraries.
